# Vendor 360 Risk Intelligence Dashboard

An end-to-end analytics dashboard designed to monitor vendor risk by combining performance, compliance, financial, and incident data into a unified scoring model.

## 🔗 Live Demo
https://vendor-risk-dashboard.streamlit.app

## 📸 Dashboard Preview
![Dashboard](dashboard.png)

## 🚀 Features
- Integrated 5 vendor datasets (performance, compliance, financials, incidents)
- Risk scoring model with categorization (Low / Medium / High)
- Identification of high-risk and critical vendors
- Top 3 risk vendor insights
- Interactive filters (Country, Category)

## 🛠 Tech Stack
- Python
- Pandas
- Streamlit
- Plotly
